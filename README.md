# SHA-1
A JavaScript implementation of the complete Secure Hash Standard SHA-1 as well as HMAC

### Installation

```bash
$ npm install sha1.js
```

### Usage

```js
var sha1 = require('sha1.js');

console.log(sha1.hex('string'));
```
